# Kalman-Filter-for-stock-price-prediction


## Kalman Filter Trading Bot

A quantitative trading research project using a **Kalman Filter** to estimate hidden market trends and generate trading signals.

The objective of this project is to explore how state-space models can be applied to financial markets for:

- price smoothing
- trend estimation
- short-term forecasting
- trading signal generation
- risk management

⚠️ This project is for research and educational purposes. It is not financial advice.

---

# Overview

Financial prices contain a large amount of noise. The observed market price is not always representative of the underlying trend.

The Kalman Filter is used to estimate the hidden market state.
